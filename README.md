## Econometrics Final Project

### 研究主題

台大學生在不同時間段 Youbike 2.0 的騎乘特徵與影響因素之分析

### 研究方法

- 不同時間段 - 每 3 個小時一組（？）：

  - **07:00 ~ 10:00** 上學 
  - 10:00 ~ 13:00
  - 13:00 ~ 16:00
  - **16:00 ~ 19:00** 放學
  - 19:00 ~ 22:00

- 研究範圍：
  - 北：科技大樓 / 南：男宿區 / 西南：水源校區 / 西北：台電大樓
  - 抓資料 - Python (Google API)
